# 評価次元: Depth of Analysis（分析の深さ）

以下のレポートを「分析の深さ」の観点で 1〜5 点で採点してください。

## ルーブリック

| スコア | 基準 |
|--------|------|
| 1 | プレスリリースや公式発表の要約にとどまる。「何が起きたか」しか書かれていない |
| 2 | 少し背景説明があるが、表面的な事実整理の域を出ない |
| 3 | 「なぜ重要か」「何が変わるか」の説明があり、読者に意味を伝えている |
| 4 | 複数の情報源を統合し、背景・影響・トレードオフを掘り下げている |
| 5 | 専門家レベルの統合・洞察。非自明な結論や、既存の知識と結びつけた深い分析がある |

## 評価の観点

- [ ] 「何が起きたか」だけでなく「なぜ重要か」が書かれているか
- [ ] 複数の情報源の知見が統合されているか
- [ ] トレードオフや反論・限界が言及されているか
- [ ] 読者が既に知っている以上の洞察が提供されているか
