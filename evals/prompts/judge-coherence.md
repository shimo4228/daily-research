# 評価次元: Coherence（構成・流暢性）

以下のレポートを「構成・流暢性」の観点で 1〜5 点で採点してください。

## ルーブリック

| スコア | 基準 |
|--------|------|
| 1 | 断片的で論理の飛躍が多い。セクション間の繋がりが不自然 |
| 2 | 全体の構成はあるが、段落間の繋がりが弱く、読みにくい箇所がある |
| 3 | セクション間の流れは自然で、論旨を追いやすい |
| 4 | 論理的な流れがあり、前の議論が後の展開を自然に導いている |
| 5 | シームレスな散文。読みやすいリズムで、全体が一つの論旨として完結している |

## 評価の観点

- [ ] 各セクションが前後と論理的に繋がっているか
- [ ] 「背景→現状→考察→展望」などの流れが自然か
- [ ] 唐突な話題転換がないか
- [ ] 結論が本文の議論から自然に導かれているか
