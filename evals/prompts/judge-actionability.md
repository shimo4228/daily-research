# 評価次元: Actionability（行動可能性）

以下のレポートを「行動可能性」の観点で 1〜5 点で採点してください。

## ルーブリック

| スコア | 基準 |
|--------|------|
| 1 | 開発アイデアや応用例が漠然としており、次のアクションが全く見えない |
| 2 | 開発の方向性は示されているが、何をすればよいか不明確 |
| 3 | 具体的なアイデアやアプリケーション例が示されており、着手の糸口がある |
| 4 | 具体的なツール・手法・アーキテクチャが提示されており、比較的すぐに着手できる |
| 5 | 即座に着手可能なレベル。具体的なツール・実装方法・注意点・参考リソースが揃っている |

## 評価の観点

- [ ] 読者が「次に何をすればよいか」を理解できるか
- [ ] 具体的なツール・ライブラリ・サービスが名指しで示されているか
- [ ] 実装上の注意点やトレードオフが言及されているか
- [ ] 参考にできるコード・ドキュメント・リソースへのリンクがあるか
