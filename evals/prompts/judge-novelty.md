# 評価次元: Novelty（テーマ斬新さ）

以下のレポートを「テーマの斬新さ」の観点で 1〜5 点で採点してください。

## ルーブリック

| スコア | 基準 |
|--------|------|
| 1 | 周知・陳腐なトピック。多くの読者が既に知っているか、古い情報 |
| 2 | 比較的新しいが、予測可能なテーマ選択。特に驚きのない内容 |
| 3 | タイムリーで読者に価値があるが、特段の独自性はない |
| 4 | 非自明なテーマ選択。見落とされがちな動向や、複数トレンドの交差点を捉えている |
| 5 | 先見性・独自性が高い。業界の次のステップや見えにくい潮流を、早い段階で捉えている |

## 評価の観点

- [ ] このトピックは今まさに注目すべき時期か
- [ ] 多くのメディアがすでに取り上げている周知の話題か
- [ ] 複数のトレンドを結びつけた独自の視点があるか
- [ ] このレポートを読んで「知らなかった」「気づかなかった」と感じる内容か
