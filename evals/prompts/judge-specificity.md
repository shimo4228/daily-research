# 評価次元: Specificity（具体性）

以下のレポートを「具体性」の観点で 1〜5 点で採点してください。

## ルーブリック

| スコア | 基準 |
|--------|------|
| 1 | 抽象的・一般的な記述のみ。固有名詞・数値・具体例がほぼない |
| 2 | 固有名詞はあるが数値や具体例が乏しい |
| 3 | 固有名詞・数値がある程度あり、主張に具体性がある |
| 4 | 豊富な固有名詞・数値・事例があり、主張を裏付けている |
| 5 | 豊富な事例・数値・比較データが揃い、主張が具体的な証拠で強力に裏付けられている |

## 評価の観点

- [ ] 具体的なツール名・企業名・プロジェクト名が登場するか
- [ ] 定量的な数値（パフォーマンス指標・市場規模・割合等）が記載されているか
- [ ] 抽象的な主張に具体例が紐づいているか
- [ ] 比較（A vs B、○○倍の改善等）があるか
